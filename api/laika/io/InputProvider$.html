  14. def forRootDirectories(roots: Seq[File], exclude: (File) ⇒ Boolean, docTypeMatcher: (Path) ⇒ DocumentType)(implicit codec: Codec): InputProvider

    Creates an InputProvider based on the specified directories, including all subdirectories.

    Creates an InputProvider based on the specified directories, including all subdirectories. The directories will be merged into a tree with a single root. If any of the specified root directories contain sub-directories with the same name, these sub-directories will be merged, too.

    roots

    the root directories of the input tree

    exclude

    the files to exclude from processing

    docTypeMatcher

    a function determining the document type based on the path of the input

    codec

    the character encoding of the files, if not specified the platform default will be used
